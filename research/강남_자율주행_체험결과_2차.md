## 1. 탑승 중

### 차량 내 구성 — 전석 (안전요원향)

전석에 모바일(카카오T 기사앱) + 랩탑(EVS_2 Driving Visualizer) 2개 기기가 설치되어 있음

#### 모바일 — 카카오T 기사앱

- 카카오T 일반 기사앱과 동일한 UI
- 남은 거리·다음 회전 정보 (예: 180m 우회전, 10m 우회전 등) 실시간 표시
- 미터기 금액 표시

#### 랩탑 — EVS_2 Driving Visualizer

별도 기술 모니터링 도구. 브라우저 기반 웹앱으로 추정 (탭 바 확인)

| 영역 | 표시 내용 |
|------|----------|
| 상단 상태바 | **녹색** = 자율주행 활성 / **노란색** = 경고 / **흰색** = 비활성 |
| 속도 | 현재 속도 (km/h) — 0 · 5 · 22 · 28 km/h 등 확인 |
| Traffic Light | 신호등 상태 아이콘 (빨강/초록 인디케이터) |
| GNSS RTK | `FIXED \| 0.2` — 고정밀 위성 측위 상태 |
| PTP Time Sync | `PC & LIDAR OK` — 센서 시간 동기화 상태 |
| Localization | `GOOD` — 위치 추정 품질 |
| Tele-Operation | 원격 개입 여부 표시 |
| Data Logging | `V` — 데이터 수집 중 |
| Route | `Gangnam-V` — 현재 운행 루트명 |
| 우측 지도 패널 | 색상별 경로 시각화 (아래 참고) |

##### 경로 시각화 색상 의미

| 색상 | 의미 |
|------|------|
| 파란 선 | 계획 주행 경로 (planned path) |
| 보라/분홍 선 | 참조 경로 (reference path) |
| 주황/빨간 점·블록 | 장애물 · 주의 구역 |
| 노란 바 (하단) | 자율주행 제한 구간 진입 경고 |

##### AUTO_PLAY 상태 알림 배너

| 배너 | 의미 | 발생 조건 |
|------|------|----------|
| 주황/분홍 배너: `이 구간에서 AUTO_PLAY가 CLOSED됩니다` | 해당 구간 자율주행 해제 예고 | 어린이보호구역 · 복잡 교차로 등 진입 전 |
| 노란 바 (지도 하단) | 제한 구간 임박 | 구간 직전 |

> 1차 체험 문서의 `자율주행 비활성` 상태와 대응하는 것으로 추정. 운전자 인터페이스에서는 `AUTO_PLAY CLOSED`로 표현됨

---

### 차량 내 구성 — 후석 (승객향)

카카오T 브랜드 탑재 태블릿 (1차 체험과 동일)

| 기기 | 표시 내용 |
|------|----------|
| 태블릿 (카카오T Safeview) | 자율주행 모드 ON/OFF 상태, 현재 속도, 주변 환경 3D 시각화, QR코드(탑승 가이드) |

##### 자율주행 모드 상태 — 후석 디스플레이 기준

| 상태 | 표시 | 특이사항 |
|------|------|---------|
| `자율주행 모드 ON` | 차량 아이콘에 파란 원형 글로우 | 신호 대기(0 km/h) 중에도 ON 유지 확인 |
| `자율주행 모드 OFF` | 파란 글로우 없음, 일반 표시 | 교차로 진입·출발 구간 등 |

##### QR코드

- 좌하단 상시 표시
- 텍스트: `탑승 가이드 확인하기`

---

### 센서 인디케이터 — 룸미러 상단

룸미러 상단 부착 기기에 `50X`(또는 유사 텍스트) + 컬러 LED(빨강·초록) 인디케이터 확인

| LED 색상 | 추정 의미 |
|----------|----------|
| 초록 | 센서 정상 |
| 빨강 | 센서 이상 또는 특정 상태 |

---

### 탑승 경험

- 야간 도심 구간 주행 (신호 대기 포함)
- 전방 택시·버스 등 차량과 근접한 상황에서 정상 주행 확인
- `AUTO_PLAY CLOSED` 알림 발생 구간에서 안전요원 수동 개입으로 전환
- 1차 체험과 마찬가지로 자율주행 on/off 전환 기준은 탑승자에게 별도 고지 없음

## 2. 전석 디스플레이 비교 — 1차 vs 2차

| 항목 | 1차 체험 (SWM 커스텀 태블릿) | 2차 체험 (EVS_2 Driving Visualizer 랩탑) |
|------|----------------------------|-----------------------------------------|
| 기기 형태 | 전용 태블릿 | 일반 랩탑 (브라우저 기반) |
| 속도 표시 | O | O |
| 센서 인디케이터 | Camera/GPS/Lidar 등 개별 | GNSS RTK, PTP Sync, Localization 통합 표시 |
| 자율주행 상태 표현 | `자율주행 실행 중` / `시작` / `비활성` 버튼 | 상단 바 색상 + `AUTO_PLAY CLOSED` 배너 |
| 경로 시각화 | 기본 지도 + 경로선 | 멀티컬러 경로·장애물 3D 시각화 |
| 신호등 정보 | O (전석 디스플레이) | O (Traffic Light 인디케이터) |
| 데이터 로깅 표시 | 없음 (확인 불가) | `Data Logging: V` 상시 표시 |
| Tele-Op 표시 | 없음 (확인 불가) | O (Tele-Operation 항목) |

> 동일 카카오T 서비스이나 차량·탑재 소프트웨어 버전에 따라 전석 인터페이스가 상이할 수 있음
